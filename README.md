# Mxnet_MobileNet

- Train
  - Download Data: `python download.py`
  - Train : `python train.py --data=mxnet_dataset --target_model=MobileNet.params`
  
- Predict: `python predict.py --model=MobileNetV1_3.params --image=image_path`
